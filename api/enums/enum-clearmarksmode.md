- [](/charting-library-docs/)- Enumerations- ClearMarksModeOn this page# Enumeration: ClearMarksMode[Charting Library](https://www.tradingview.com/charting-library-docs/latest/api/modules/Charting_Library).ClearMarksMode

Mode to clear the marks on the chart.

## Enumeration Members[​](#enumeration-members)
### All[​](#all)
• All = `0`

Will clear both bar marks AND timescale marks - default value

### BarMarks[​](#barmarks)
• BarMarks = `1`

Will only clear bar marks

### TimeScaleMarks[​](#timescalemarks)
• TimeScaleMarks = `2`

Will only clear timescale marks
