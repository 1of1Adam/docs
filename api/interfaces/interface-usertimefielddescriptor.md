- [](/charting-library-docs/)- Interfaces- UserTimeFieldDescriptorOn this page# Interface: UserTimeFieldDescriptor[Charting Library](https://www.tradingview.com/charting-library-docs/latest/api/modules/Charting_Library).UserTimeFieldDescriptor

Used in the schema defined in exportData API to describe the user time field.
This is used when `includeUserTime: true` is defined in `exportData` to add user time (aka time that is displayed to the user on the chart) to exported data.
