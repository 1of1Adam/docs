## Properties[​](#properties)
### displayCounterInTab[​](#displaycounterintab)
• `Optional` displayCounterInTab: `boolean`

Controls whether the number of items is displayed on a custom page. Set to `true` to display the count, or `false` to hide it.

### id[​](#id)
• id: `string`

Unique identifier of a page

### tables[​](#tables)
• tables: [`AccountManagerTable`](https://www.tradingview.com/charting-library-docs/latest/api/interfaces/Broker.AccountManagerTable)[]

It is possible to display one or more tables in this tab.

### title[​](#title)
• title: `string`

Page title. It is the tab name.
