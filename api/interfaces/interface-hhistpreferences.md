- [](/charting-library-docs/)- Interfaces- HHistPreferencesOn this page# Interface: HHistPreferences[Charting Library](https://www.tradingview.com/charting-library-docs/latest/api/modules/Charting_Library).HHistPreferences

Histogram Preferences

## Properties[​](#properties)
### colors[​](#colors)
• colors: `string`[]

Colors for the histogram up and down bars.
Example:
```
colors: ['#1592e6', '#fbc123']
```

### direction[​](#direction)
• direction: [`HHistDirection`](https://www.tradingview.com/charting-library-docs/latest/api/enums/Charting_Library.HHistDirection)

Direction of the histogram.
Whether the histogram will be shown on the left or right edge of the chart

### percentWidth[​](#percentwidth)
• percentWidth: `number`

Percentage width of the chart for which the histogram should occupy

### showValues[​](#showvalues)
• showValues: `boolean`

Should the values be shown

### transparencies[​](#transparencies)
• transparencies: `number`[]

Transparency values for the histogram

### valuesColor[​](#valuescolor)
• valuesColor: `string`

Color for values shown

### visible[​](#visible)
• visible: `boolean`

Should the histogram be visible
